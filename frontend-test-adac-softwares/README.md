## Observações técnicas

Durante a instalação das dependências, o `npm audit` aponta vulnerabilidades
em dependências internas do Next.js e ESLint.

Essas vulnerabilidades não afetam o funcionamento da aplicação neste contexto,
não são exploráveis no escopo do projeto (landing page estática)
e foram mantidas para preservar a estabilidade e compatibilidade da stack.
